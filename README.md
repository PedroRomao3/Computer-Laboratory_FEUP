# LCOM-Blackjack

## Final Grade: 19,8
This repository holds the project for the Computer laboratory course. A course where students are asked to develop low level C code to interact with various hardware peripherals.
The [Teaching Assistance](/TA) folder holds some of the content I created as a TA for the course, the content I am allowed to share.
## Devices implemented
- Mouse
- Keyboard
- Graphics
- UART serial port
- RTC
- Timer
